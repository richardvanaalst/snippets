SASS & Compass

Stop watching

CTRL-C

SASS

Watch a file

sass --watch input.scss:output.css

Watch a folder

sass --watch dir/scss:dir/css

Compass

Create project

compass create project_name

Watch a project

compass watch project_dir

Git

Base

git clone
git pull
git push
git status
git diff

Initialize an (empty) repository

cd directory/of/project
git init
git add .
git commit -a -m "Message"

Commit -all -message

git commit -a -m "Message"

Switch branch

git checkout branchname

Subversion

Base

cd directory/of/checkout
svn co repository_url directory_url
of 
svn checkout repository_url directory_url
svn up
of 
svn update
svn cleanup --non-interactive
svn status
svn diff

Add all recursively

svn add --force *

Commit with message

svn ci -m "Message"
of 
svn commit -m "Message"

Merge

svn merge url
Merge from trunk to branch
svn merge ^/trunk
Merge from revision XYZ to HEAD
svn merge -rXYZ:HEAD
Merge changeset (revision) XYZ only
svn merge -c XYZ

Switch

svn sw url
